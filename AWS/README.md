# To configure the aws credentials 

[AWS CLI](https://aws.amazon.com/cli/)

> Download the AWS CLI depending on your Operating System.

    aws configure
**Enter** 
* *aws_access_key_id* 
* *aws_secret_access_key*


## 1. Windows manual 
    %UserProfile%\.aws\credentials

**edit** *credentials file* 

    [default]
    aws_access_key_id = 
    aws_secret_access_key =

## 2. MacOS and Linux

    ~/.aws/credentials
